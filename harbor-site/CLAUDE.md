# 合同会社harbor — コーポレートサイト 設計書

> 内部資料。公開ディレクトリ（dist/）には含めない。

## 1. コンセプト
- **目的**：信頼感・名刺代わり／商品・サービスの紹介
- **ターゲット**：YouTube運用に興味がある toC 企業
- **トーン**：高級・上品、黒でシック
- **キーメッセージ**：「選ばれるチャンネルを、戦略から。」
- 社名 harbor（＝港）にちなみ、「ブランドが戻ってくる拠り所」という世界観を余白とゴールドで上質に表現。

## 2. カラートークン（:root）
| 変数 | HEX | 用途 |
|------|------|------|
| `--ink` | `#0B0B0D` | ベース背景（黒） |
| `--ink-2` | `#15151A` | パネル・カード |
| `--ink-3` | `#1F1F26` | ホバー・境界面 |
| `--cream` | `#F4F1EA` | メインテキスト／反転セクション背景 |
| `--gold` | `#BFA06A` | 差し色（上品なゴールド） |
| `--gold-soft` | `#D8C39A` | ゴールドの明色 |
| `--muted` | `#8C8B90` | 補助テキスト |
| `--line` | `#2A2A31` | 罫線・区切り |

コントラスト：本文 cream(#F4F1EA) on ink(#0B0B0D) ≒ 17:1（AAA）。gold on ink ≒ 6.6:1（AA以上）。

## 3. フォント（Google Fonts）
- 見出し：`Cormorant Garamond`（欧文セリフ・600/700）＋ `Shippori Mincho`（和文明朝・500/700）
- 本文／UI：`Jost`（欧文サンセリフ・300/400/500）＋ `Noto Sans JP`（和文・300/400/500）
- 和文はシステムフォント任せにせず、必ず Google Fonts を指定。

## 4. レイアウト方針
- 余白広め（セクション上下 clamp(6rem, 12vh, 10rem)）。
- 最大幅 1180px のコンテナ。装飾は最小限、細い罫線とゴールドのアクセントのみ。
- ファーストビュー（ヒーロー）は JS 非依存で即表示。
- 写真は images/ にローカル保存。世界観統一のため `filter: grayscale(1)` ＋暗いオーバーレイ。デモ画像（SVG）を後から差し替え可能。

## 5. ページ構成（複数ページ）
1. `index.html` — ヒーロー／サービス概要／選ばれる理由・強み／実績数字／お客様の声（抜粋）／代表挨拶（抜粋）／CTA
2. `service.html` — サービス3本（YouTube運用代行・LINE運用/構築・YouTubeコンサル）詳細＋選ばれる理由・強み
3. `about.html` — 会社概要・店舗情報／代表挨拶（全文）
4. `voice.html` — お客様の声（全件）
5. `contact.html` — お問い合わせ（Netlify Forms＋電話＋メール）
6. `thanks.html` — 送信完了ページ

共通：ヘッダー（ロゴ文字＋ナビ＋ハンバーガー）、フッター（連絡先・コピーライト）。

## 6. 各セクション仕様
- **ヘッダー**：固定（sticky）、スクロールで背景濃く。SP はハンバーガー（aria-expanded 連動、ESC/オーバーレイで閉じる）。
- **ヒーロー**：全画面手前、暗いビジュアル＋大見出し（明朝）＋サブコピー＋CTA 2本。
- **サービス**：3カード（番号・タイトル・説明・リンク）。
- **強み**：3〜4項目、番号付きリスト＋細罫線。
- **実績数字**：3指標（カウントは静的表示／※デモ値）。
- **お客様の声**：引用カード（業種ラベル付き／※デモ）。
- **代表挨拶**：ポートレート＋メッセージ＋署名。
- **CTA**：反転（cream背景）or 罫線囲み、フォームへ誘導。
- **フッター**：社名・連絡先（tel/mailto）・ナビ・コピーライト。

## 8. フォーム（Netlify Forms）
- `<form name="contact" method="POST" data-netlify="true" netlify-honeypot="bot-field" action="/thanks.html">`
- hidden `form-name`、honeypot `bot-field`、必須項目に required、ラベル紐付け。

## 9. アクセシビリティ
- :focus-visible でフォーカスリング可視（ゴールド）。
- prefers-reduced-motion: アニメ・スムーススクロール無効化。
- すべての img に alt。装飾SVGは aria-hidden。
- スキップリンク、ランドマーク（header/main/footer/nav）。
