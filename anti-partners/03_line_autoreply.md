# 💬 公式LINE 自動返信設計

新規友達追加時の自動応答メッセージの設計書です。
LINE公式アカウントの「あいさつメッセージ」「応答メッセージ」に設定してご利用ください。

トーンの基本：**誠実・温かい・急かさない。** 営業色を出さず、まず安心していただくことを優先します。

---

## 1. 友達追加時のあいさつメッセージ（自動送信）

友達追加された瞬間に自動で送られるメッセージです。
「歓迎」→「紹介者ヒアリング」→「無料相談CTA」の順で構成します。

```
はじめまして。
退職給付金サポート「アンタイ」です。
数ある中から見つけてくださり、ありがとうございます。

退職の前後でお受け取りいただける給付金について、
あなたのご状況に合わせてご案内しています。

まずは、ひとつだけ教えてください。

──────────────
🔰 ご紹介者のお名前
──────────────
「○○さんの紹介です」と
メッセージでお送りいただけますか?

ご紹介経由の方は、ご案内をスムーズに
お繋ぎできるようにしております。

※ ご紹介者がいない場合は
　「直接見つけました」とお送りください。
　もちろん大歓迎です。
```

> 🗒 設計意図
> 最初の1通で「紹介者名のヒアリング」を完了させ、誰からの紹介かを土方が即座に把握できるようにします。
> 紹介者がいない流入も拾えるよう、逃げ道（「直接見つけました」）を用意しています。

---

## 2. 紹介者名を受け取った後の応答メッセージ

お客様が名前を送ってくださったら、無料相談へ自然に誘導します。
（キーワード応答が難しい場合は、土方が手動で返信する想定の文面としても使えます）

```
ありがとうございます。
○○さんからのご紹介ですね。承りました。

ここから先は、担当の土方が
直接ご案内させていただきます。

退職給付金は、ご状況によって
受け取れる金額や条件が変わります。
まずは無料で、あなたが対象になるか・
いくらくらい受け取れそうかをお調べします。

──────────────
🎁 無料相談（オンライン / LINE）
──────────────
「相談したい」とお送りいただければ、
日程の候補をご案内します。

・しつこい営業は一切いたしません
・相談だけで終えていただいても大丈夫です
・ご質問だけのご連絡も歓迎です

どうぞお気軽に、ひと言お送りください。
```

> 🗒 設計意図
> 「相談だけでOK」「営業しない」を明示し、心理的ハードルを下げます。
> 主導権はお客様に置いたまま、次の一歩（「相談したい」と送る）だけを示します。

---

## 3. キーワード応答の設定例

LINE公式アカウントの「応答メッセージ（キーワード設定）」に登録する想定です。

| キーワード | 返す内容 |
| --- | --- |
| `相談` / `相談したい` | 無料相談の日程候補をご案内する旨を返信 → 以降は土方が手動対応 |
| `料金` / `費用` | 「ご相談・対象診断は無料です」と案内 |
| `対象` / `対象になる?` | 「ご状況をお伺いした上で土方が判断します」と案内 |
| `紹介` | 紹介者名のヒアリング文（あいさつメッセージと同様）を再送 |

> ⚠️ 注意
> 自動応答はあくまで「最初のひと言」までです。
> 個別のご状況に関わるやり取りは、必ず土方が手動で引き取ります。
> （金額や対象可否を自動返信で断定しないこと）

---

## 4. 運用メモ

```
✅ あいさつメッセージは「紹介者名ヒアリング」を最優先
✅ 紹介者がいない流入も取りこぼさない導線にする
✅ 無料相談CTAは「相談だけでOK」を必ず添える
✅ 金額・対象可否の判断は自動化せず、土方が手動対応
✅ 対応時間：平日 10:00〜23:00 / 土日祝も対応（遅れる場合あり）
```

---

## 5. 自動返信に添える対応時間の一文（任意）

営業時間外に追加された方への配慮として、末尾に添えても良い一文です。

```
※ ご返信は、平日10:00〜23:00を目安に
　順次させていただいております。
　土日祝も対応しておりますので、
　お返事まで少しお時間をいただく場合がございます。
```
