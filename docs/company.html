ABOUT ANTAI

「もらえるはずの安心」を、
取りこぼさない社会へ。

制度は、知っている人だけが得をする——。私たちはその不公平を、なくしたい。
アンタイは、社会保険給付金の正しい申請を通じて、退職という人生の転機を支えます。

MISSION

退職後の生活を、
「安泰」に。

退職は、誰にとっても大きな決断です。次の一歩を踏み出すとき、お金の不安が足かせになってはいけない。

失業保険や傷病手当金は、働く人のために国が用意した正当な権利です。しかし制度は複雑で、知らないまま・申請ミスのまま、本来受け取れる給付を取りこぼしてしまう人が後を絶ちません。私たちは、その「もったいない」をなくすために存在します。

Vision

誰もが、制度を「正しく・公平に」使える社会へ。情報格差をなくし、安心して次の挑戦に進める世の中をつくる。

Value

誠実 / 透明 / 正確。できることとできないことを正直に伝え、不正のない適正な申請だけを、確実にやり遂げる。

MESSAGE

代表メッセージ

社会保険給付金のサポート業界には、残念ながら「給付金を増やせる」「必ずもらえる」といった、誤解を招く宣伝が少なくありません。2025年には国民生活センターが注意喚起を行うなど、利用者が不安を抱く状況が生まれています。

私たちアンタイは、その流れに真っ向から異を唱えます。給付額は法律で決まっており、増やすことはできません。私たちにできるのは、あなたが本来受け取れる給付を、正確に、漏れなく申請すること。そして、できないことは「できない」と正直に伝えることです。

不安につけ込むのではなく、正しい知識で不安を解消する。それが、退職という転機に寄り添う私たちの仕事だと信じています。あなたの「安泰」を、誠実に支えます。
